  • background-repeat

    background-repeat:
    Value = repeat (defolt Повторяет кратинку на фоне)
    Value = no-repeat (Запрещает повторение и прижимает к левому верхнему углу)
    Value = repeatX (Повторяет но только по оси Х)
    Value = repeatY (Повторяет но только оси У)

  • Background-position

    Задает позиционирование фона относительно родителя
    Значение можно задать как в px так %
    Пример: background position: 20% 20%
